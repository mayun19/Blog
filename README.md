# Blog by Nuya

## Demo

Access site at https://blog-bynuya.vercel.app/

## 🚀 Getting Started

Clone this repo

```bash
  git clone git@github.com:mayun19/Blog.git

```

Install Blog by Nuya with npm

```bash
  cd Blog
  npm install
```

Runs the app in the development mode.

```bash
  npm run dev
```
